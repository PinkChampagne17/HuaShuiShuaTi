# HuaShuiShuaTi.App

划水刷题是一款Material Design风格的刷题web应用，使用Angular 8开发而成。

只建议你使用该应用，不建议通过本应用学习代码（意味深）。

## 支持的语言

## 如何在本地开发并运行？

在运行之前，请先确保电脑上已经安装了以下几个组件:
- Git
- npm
- Angular cli

同步依赖项：
```
npm install
```

安装Angular cli:
```
npm install -g @angular/cli
```

使用Angular cli运行：
```
ng serve --open
```
