## 如何在本地运行？

在运行之前，请先确保电脑上已经安装了以下几个组件:
- Git
- npm
- Angular cli

同步依赖项：
```
npm install
```

安装Angular cli:
```
npm install -g @angular/cli
```

使用Angular cli运行：
```
ng serve --open
```
